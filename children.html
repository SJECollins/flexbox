order:

Applied to children.

Sets order flex items appear in container.

.parent: {
  display: flex;
}
.flex-items {
  min-height: 80px;
  width: 40px;
  order: _____;
}

Flex items display in order by flow of DOM elements.

flex-basis:

Applied to children.

Sets initial size of flex item, before space is distributed by other flex factors.

An auto value sizes the item to its size property.

.parent: {
  display: flex;
  width: 180px;
}
.flex-items {
  min-height: 80px;
  width: auto;
  flex-basis: _____;
}

align-self:

Applied to children.

Sets alignment of individual flex items.

Accepts the same values as align-items.

Can override align-items.

Default is stretch.

.parent: {
  display: flex;
  height: 160px;
  width: 180px;
}
.flex-items {
  min-height: 80px;
  width: 40px;
  align-self: _____;
}
