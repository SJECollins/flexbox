order:

Applied to children.

Sets order flex items appear in container.

.parent: {
  display: flex;
}
.flex-items {
  min-height: 80px;
  width: 40px;
  order: _____;
}

without order

Flex items display in order by flow of DOM elements.

One
Two
Three

order: 1 / 2 / 3, etc

Applying order: 1 to box three, order: 2 to box one and order: 3 to box two.
